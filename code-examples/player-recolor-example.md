# Player Recolor Example

Recolors of player skins are set up in Core Keeper using Scriptable Data, you can preview an example by opening up the Scriptable Data Editor Window, selecting Core Keeper Assets (readonly) as the directory and sorting by Skin Base / Replacement Color.&#x20;

These replacement colors target source colors, which are the colors that we will be replacing on the player character.&#x20;

The array size of the replacement colors must match the array size of the source colors.&#x20;
