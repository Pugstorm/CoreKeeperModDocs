# ModCommands Example

Console commands use the Quantum Console package internally, to find out more about Quantum Console and its' features that you can use to customize the console further, you may check out the documentation [here](https://www.qfsw.co.uk/docs/QC/articles/quickstart/quickstart.html).

You may enable Quantum Console in-game by building a mod which sets the bool `Manager.enableConsole = true`.

```csharp
using PugMod;
using UnityEngine;

public class EnableConsole : IMod
{
    public void EarlyInit()
    {
        // This enables Console Commands.
        Manager.enableConsole = true; // Make sure to run this in EarlyInit()!
    }

// ...Init, ModObjectLoaded(...), Shutdown(), Update()...
}
```

The below example showcases how you can create a new command that will appear in Quantum Console when opened in-game. This particular command uses the `modding.` prefix, which is similar to the `player.` prefix which the base game uses for some commands.&#x20;

Using a prefix is completely optional.

The below example command can be triggered in-game by opening Quantum Console using `Ctrl+x` if the console has been enabled, and typing in `modding.playPuffVFX 5 20` , where 5 is the puffID and 20 is the amount of particles that will spawn.

```csharp
using PugMod;
using QFSW.QC;
using UnityEngine;
using UnityEngine.Scripting;

public class MyNewCommands : MonoBehaviour
{
    [CommandPrefix("modding.")]
    public static class ModdingCommands
    {
        [Preserve]
        [CommandWithModSupport("playPuffVFX", "Plays a visual Puff effect at the players' position.", paramsInGlobalSuggestions: 1)]
        public static void PlayPuffVFX(PuffID puffID, int particleCount = 10)
        {
            if (API.Client.LocalPlayer == null)
            {
                Manager.menu.quantumConsole.LogToConsole("You must be in-game to use this command.");
                return;
            }

            Vector3 playerPosition = API.Client.LocalPlayer.transform.position;
            API.Effects.PlayPuff((int)puffID, playerPosition, particleCount);
            Manager.menu.quantumConsole.LogToConsole($"Played visual effect with ID: {puffID} and particle amount: {particleCount}");
        }
    }
}
```
