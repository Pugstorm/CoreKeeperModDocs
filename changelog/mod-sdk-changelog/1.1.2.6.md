# 1.1.2.6

* Upgraded the Unity engine version from 2022.3.62f2 to 6000.0.58f2. You can download the new version in the Unity 6 > LTS section in the download archive: [https://unity.com/releases/editor/archive](https://unity.com/releases/editor/archive). If you are having issues after upgrading, try closing the editor and delete the project’s Library folder before reopening.
* Removed the obsolete Visual Studio Code Editor package. The Visual Studio Editor package now includes support for VS Code.
* Removed the OpenProject scripts. Instead, you should open the project using the [ Unity Hub](https://unity.com/unity-hub) available for download on Unity’s website.
