# 0.7.3

* Upgraded the Unity engine version from 6000.0.58f2 to 6000.0.59f2. This was required for fix a frequent source of crashes, and it is not expected to impact modding. You can download the new version in the Unity 6 > LTS section in the download archive: [https://unity.com/releases/editor/archive](https://unity.com/releases/editor/archive). If you are having issues after upgrading, try closing the editor and delete the project’s Library folder before reopening.
