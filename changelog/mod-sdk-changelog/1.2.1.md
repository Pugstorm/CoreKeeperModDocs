# 1.2.1

* CraftingAuthoring now uses a `ModdedObjectID` string when ObjectID is set to ObjectID.None, which’ll be converted to an ObjectID during runtime. You may use this change to add modded items to workbenches and such.&#x20;
* All Graphical Prefabs which have a component that inherits from IPoolable will now automatically be pooled. In order to adjust the size of pooling or to disable it you may create a `PooledModGraphicalObjectBank` Scriptable Object via the Create menu.&#x20;
* The Sprite Render on the Mouse cursor has been upgraded to a SpriteAsset which can now be modded via the ScriptableDataBlock editor.&#x20;
* QuantumConsole has now been exposed to mod developers and can be switched on by setting MenuManager.enableConsole to true. It can then be opened in-game via ctrl+x. The object suggestor supports modded Object Names when using the spawnByName command.
