# 1.1.2.5

* Upgraded the Unity engine version from 2022.3.50f1 to2022.3.62f2. You can download the new version in the Unity 2022 > LTS section in [the download archive](https://unity.com/releases/editor/archive). If you are having issues after upgrading, try closing the editor and delete the project’s Library folder before reopening. This upgrade is motivated by the security vulnerability found in older versions of the Unity engine. For more information, see [Unity’s official announcement](https://unity.com/security/sept-2025-01).
* Addressables is now included as an embedded package in the SDK.
