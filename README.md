# Welcome

Welcome to the official documentation for the Core Keeper modding SDK. This documentation is your go-to spot for everything Core Keeper modding. Be it how to create, share, or find mods, we have information for it all. Additionally you will also find updates on the latest changes to the mod SDK and code examples so you can easily get going.&#x20;

Modding is way more fun when you're not going at it alone. If you ever get stuck, need a new idea, or you just want to connect with other modders and developers, our [Official Discord](https://discord.gg/CoreKeeper) community is super open and welcoming!

We care about what you think. Whether you find something we can do better, have a cool idea, or want to point out if something is/isn't working, feel free to reach out. Your input helps us keep the modding community lively and growing right alongside the game itself.

# How to Contribute

Thanks to everyone pitching in to make the Core Keeper modding documentation and tools better. If you spot something confusing, want to flesh out a guide, or just have a smart idea to help other creators, we are always happy to see new contributions.

## Contributing on GitHub

We manage all contributions through **GitHub pull requests**.

Here is how it works:

1. **Create a fork.** Make your own copy of the repo so you can edit it freely.
2. **Make your changes**. Tweak or add whatever you think improves things. Keep them clear and focused so it's easy for us to review them.
3. **Open a pull request.** When you're ready, submit a pull request back to the main repo. Our team will review it, give feedback if needed, and merge it once everything looks good and ready.

Every contribution helps us keep the documentation accurate, clear, and actually useful for the entire modding community. So seriously, a big thank you for taking the time to support it!
