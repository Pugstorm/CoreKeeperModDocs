# Explore Mods

Core Keeper mods can be found on either Steam Workshop or Mod.IO.&#x20;

{% hint style="info" %}
Note that Steam Workshop is in beta and it is expected that it will be missing some mods which may exist on mod.io
{% endhint %}

## Mods on Steam Workshop

{% embed url="https://store.steampowered.com/app/1621690/Core_Keeper/" %}

Head over to Core Keepers' Steam Workshop page and use tags to sort through all of the available mods.&#x20;

{% hint style="warning" %}
To use Steam Workshop you will need to own Core Keeper on Steam.
{% endhint %}

## Mods on mod.io

{% embed url="https://mod.io/g/corekeeper" %}

Head over to Core Keepers' mod.io page and use tags to sort through all of the available mods.&#x20;
